# **App Name**: TradeView

## Core Features:

- Cryptocurrency Listings: Display of featured cryptocurrencies with current prices and percentage change.
- Portfolio Balance Display: Overview of the user's portfolio balance.
- Simulated Buy Crypto Button: Button that simulates buying cryptocurrency.
- Simulated Quick Trade Button: Button that simulates quick trades of cryptocurrency.
- AI-Powered Market Insights: Use an AI tool to provide market insights, highlighting significant trends and news related to selected cryptocurrencies.
- Bottom Navigation Bar: A set of navigational elements at the bottom of the screen to allow intuitive traversal between sections of the application.

## Style Guidelines:

- Primary color: Saturated violet (#9400D3) for an innovative, modern aesthetic.
- Background color: Light desaturated violet (#F0E5F5) for a clean and unobtrusive background.
- Accent color: Slightly lighter, more saturated violet (#A020F0) to highlight interactive elements.
- Body and headline font: 'Inter' sans-serif with a modern and neutral feel. Note: currently only Google Fonts are supported.
- Use of rounded corners for all containers, cards, and buttons.
- Simple line icons for navigation and actions.
- Subtle transitions for loading and data updates.
